Backup

This module provides functionality to copy files or directories from a source to a destination. It supports various options for handling conflicts and symlinks.

Options

  • source: The file or folder to copy.
  • dest: The destination to copy to.
  • strategy: (Optional) Strategy for managing conflicts. Can be "replace" or "merge" (default: "merge").
  • symlink: (Optional) How to handle symlinks. Can be "deep", "shallow", or "skip" (default: "skip").

Example


  copy({
 source = "./folder",
 dest = "./copy"
  })
